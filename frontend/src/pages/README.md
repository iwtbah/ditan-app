# Pages 层说明

`src/pages` 是路由页面承接层。

职责：

- 作为 `app/router.tsx` 的稳定路由入口
- 承接路由参数、页面级元信息和路由级布局决策
- 组合对应 feature 的页面级输出

约束：

- 页面层不直接写请求
- 页面层不保存服务端数据
- 页面层不承载复杂业务编排
- 若当前页面只是 feature 的薄包装，也应保持为显式页面组件，而不是长期停留在纯 re-export 形态

当前状态：

- 本项目仍处于工程结构收口阶段
- 部分页面暂时只承担“路由承接 + feature 转发”职责
- 等真实接口和页面级元信息接入后，再逐步把路由参数解析、标题等职责补到页面层
