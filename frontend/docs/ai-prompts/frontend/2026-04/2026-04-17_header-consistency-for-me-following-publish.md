# Header 一致化优化

- 日期：2026-04-17
- 来源范围：我的页、关注页、新建笔记页、首页、搜索页 Header
- 标签：header、ui、一致性、移动端、最小改动

## 原始提示词

我的页header两边有空隙，跟迪探页header不一样，并将新建笔记页，关注页headear风格一致化

把首页、搜索页 Header 也收进同一骨架

## 上下文说明

- 目标是继续沿用既有的两层 Header 体系，不改业务逻辑、不改页面信息架构。
- 本轮只允许做 Header 样式与轻结构统一，不顺手改未提及页面。
- 统一参照页是迪探页当前使用的系统级 `PageHeader` 风格。
- 首页与搜索页允许通过共享骨架的插槽能力承载搜索入口，但不新增功能、不改变交互语义。
