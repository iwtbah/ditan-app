# 初始化落地阶段的前端基础代码需求提示词

- 记录日期：2026-04-13
- 归档范围：frontend
- 主题标签：initialization, app-entry, router, query-client, zustand, api-client, page-skeleton, tailwind
- 来源说明：用户要求基于既定目录结构完成初始化落地代码

## 原始提示词

基于你刚才给出的目录结构，现在进入“初始化落地”阶段。

请继续完成以下内容：
1. 生成 main.tsx、App.tsx 的基础结构
2. 生成 router/index.tsx 或 router 配置文件
3. 生成 queryClient 初始化代码
4. 生成 Zustand 的基础 store 示例（如 appStore、authStore）
5. 生成 api 请求封装基础层
6. 生成 types 示例
7. 生成首页、店铺详情、笔记详情、登录页的页面骨架占位组件
8. 配置 Tailwind CSS 的基础全局样式规范

要求：
- 代码按工程化最佳实践组织
- 不要先写复杂业务逻辑
- 所有示例文件都要能直接作为项目起步代码
- 页面先输出模块骨架和占位文案
- 在代码注释中解释每个文件在项目中的职责
